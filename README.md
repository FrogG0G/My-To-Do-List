# My To-Do List App

Простое и функциональное приложение для управления задачами с поддержкой локализации, системой приоритетов, поиском, фильтрацией и сортировкой.  
Все данные сохраняются в базе данных и не теряются после перезапуска приложения.

---

## 📌 Описание проекта

Приложение позволяет:
- Создавать задачи
- Просматривать список задач
- Редактировать и удалять задачи
- Отмечать задачи как выполненные
- Отслеживать статистику невыполненных задач по приоритету

Интерфейс автоматически поддерживает светлую и тёмную тему в зависимости от системных настроек устройства пользователя.


---

# 🖥 Структура приложения

## 1️⃣ Главный экран

Главный экран является точкой входа в приложение.

На нем расположены:
- Кнопка **Настройки**
- Кнопка **Добавить задачу**
- Кнопка **Просмотр задач**
- Блок **Статистики**

Блок статистики отображает количество **невыполненных** задач по каждому приоритету.

После отметки задачи как выполненной она автоматически исключается из статистики.

## 3️⃣ Создание задачи

При нажатии на кнопку **Добавить задачу** открывается экран создания задачи.

Поля:
- Название
- Описание
- Дата выполнения
- Приоритет

После сохранения:
- Задача записывается в базу данных
- Она появляется в списке задач

## 4️⃣ Экран просмотра задач

Экран содержит список всех созданных задач.

Доступные функции:

### 🔍 Поиск
Поиск задач по названию.

### 🎯 Фильтрация
Отображение задач определенного приоритета.

### ↕ Сортировка
Сортировка:
- По алфавиту
- По дате
- По приоритету (по возрастанию и по убыванию)

## 5️⃣ Работа со статусом задачи

В списке задач слева от каждой задачи расположен чекбокс.

Если отметить задачу:
- Она получает статус "Выполнена"
- Карточка задачи становится серой
- Текст зачеркивается
- Задача исключается из статистики

Статус сохраняется в базе данных.

---

## 6️⃣ Детальный просмотр задачи

При нажатии на задачу открывается экран подробной информации.

Отображается:
- Название
- Описание
- Дата
- Приоритет

Доступно:
- Редактирование
- Удаление

Все изменения сохраняются в базе данных.

# 🗄 Хранение данных

Все задачи хранятся в базе данных приложения.  
Это обеспечивает:

- Сохранность данных между перезапусками
- Корректную работу поиска, фильтрации и сортировки
- Актуальность статистики
